## 新建项目
npx create-nuxt-app ProjectName
[参考-nuxt官方文档](https://www.nuxtjs.cn/guide/installation)
![示例图片.png](https://yanxuan.nosdn.127.net/86bd91ceaaa8955d1293848bfd46d05b.png)

## To get started:
```bash
cd ProjectName
npm run dev
```
## To build & start for production:
```bash
cd ProjectName
npm run build
npm run start
```
